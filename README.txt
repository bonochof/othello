○起動方法
　Othello.exeを実行する．

●操作説明
　1. タイトル画面
　　　キーボード入力で次に進みます．
　2. 難易度(深さ)選択画面
　　　マウス入力で難易度を選択します．
　3. モード選択画面
　　　マウス入力でモードを選択します．
　　　モードは「通常モード」「ハンデモード」「最弱モード」「革命モード」があります．(後述)
　4. スタート画面
　　　「START」をクリックするとゲーム開始です．
　5. メインゲーム画面
　　　マウス入力で石を置きます．
　　　ユーザは先行(黒)です．
　　　
○モード
　1. 通常モード
　　　通常ルールのオセロです．
　2. ハンデモード
　　　四つ角をコンピュータが取った状態から始まります．
　3. 最弱モード
　　　ゲーム終了時に石の少ない方が勝ちとなります．
　4. 革命モード
　　　角を取ると，他の角に置かれていた石が反転します．
